## ⚙️ Environment Variables and Deployment

- **Environment Variable:**  
  You must set your `OPENAI_API_KEY` as an environment variable to run this project.
  
- **Deployment Options:**  
  This project can be deployed on any platform that supports Python web apps, such as Heroku, DigitalOcean, or AWS. For portfolio purposes, even running it locally and capturing screenshots/gifs is sufficient.
